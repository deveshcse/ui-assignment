# Aventisia Frontend

A modern, responsive Knowledge Base management system.

## 🌐 Live Demo
[https://ui-assignment-plum.vercel.app/](https://ui-assignment-plum.vercel.app/)

## 💻 GitHub Repository
[https://github.com/deveshcse/ui-assignment.git](https://github.com/deveshcse/ui-assignment.git)

## ✨ Key Features

- **Dynamic Knowledge Base Management**: Create and manage multiple knowledge bases with a responsive grid layout.
- **Intelligent Search**: Real-time filtering of knowledge bases via a global search bar.
- **Seamless Creation Flow**: Dedicated "Create New" modal with automatic recording of creation dates.
- **Responsive Navigation**: A fully adaptable sidebar and header system designed for both desktop and mobile users.
- **Theme-Aware UI**: Native support for light and dark modes using a consistent design system and theme tokens.
- **Clean Codebase**: Optimized component structure with a focus on maintainability and removal of redundant template code.

## 🛠️ Technology Stack

- **Framework**: [React](https://reactjs.org/) + [Vite](https://vitejs.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **UI Components**: [Shadcn UI](https://ui.shadcn.com/) (Radix UI)
- **Icons**: [Lucide React](https://lucide.dev/)

## 🏗️ Project Structure

- `src/components/knowledge-base`: Core Knowledge Base logic, cards, and creation modals.
- `src/components/layout`: Global layout components including Header and Sidebar.
- `src/components/ui`: Reusable primitive UI components built with Shadcn.
- `src/index.css`: Global styles and theme variable definitions.

## 🏁 Getting Started

1. **Clone this repository**

```bash
git clone https://github.com/deveshcse/ui-assignment.git
```   

2. **Install Dependencies**:
   ```bash
   npm install
   ```

3. **Run Development Server**:
   ```bash
   npm run dev
   ```

4. **Build for Production**:
   ```bash
   npm run build
   ```
